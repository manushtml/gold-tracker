# 台銀黃金存摺價格追蹤 Line Bot

每日下午 3 點自動發送台灣銀行黃金存摺賣出價格通知，支援 Line 群組動態調整目標買進價格。

## 功能

- 每日下午 3 點自動推播台銀黃金存摺賣出價
- 與昨日收盤價比對，顯示漲跌幅
- 達到目標買進價格時發出特別提醒
- 支援 Line 群組指令動態調整目標價格

## Line 群組指令

| 指令 | 說明 |
|------|------|
| /price 或 查詢金價 | 查詢台銀即時賣出價 |
| /status 或 查詢狀態 | 查詢目標價與當前狀態 |
| /history 或 查詢歷史 | 查詢近 5 日收盤價 |
| /report 或 今日報告 | 取得完整每日報告 |
| /set_price 2550 | 設定目標買進價格為 2550 元 |
| /help 或 說明 | 顯示指令說明 |
